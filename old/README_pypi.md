## Keyboard Shortcuts
- **Play / Pause:** `Spacebar`
- **Back:** `Left Arrow`
- **Forward:** `Right Arrow`
- **Frame Back:** `<` (`,`)
- **Frame Forward:** `>` (`.`)
- **Restart:** `R`
- **Volume Up:** `Up Arrow`
- **Volume Down:** `Down Arrow`
- **Mute / Unmute:** `M`
